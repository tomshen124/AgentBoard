# AgentBoard Brand Direction

This folder keeps working brand references for the desktop client.

## Current Direction

- Product name: AgentBoard
- Kernel name: TaskLoop
- Visual idea: knowledge workbench, layered context, local task loop
- Palette: graphite, warm paper, sage, amber
- Tone: focused, durable, local-first, assistant-like without feeling decorative

## Assets

- `agentboard-knowledge-workbench-concept.png`: preferred logo and icon exploration for the knowledge companion direction.
- `agentboard-loop-mark-concept.png`: alternative mark exploration with a stronger task-loop symbol.

The app currently uses an in-client CSS brand glyph for the sidebar and home surface. Production app icons should be exported from a final vector source before replacing `build/icon.png` and `resources/icon.png`.
